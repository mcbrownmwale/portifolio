Investigating Covid-19 Virus Trends (January 2020 - July 2020)

Tools: Python

1. Project Overview

This project analyzes the early trends of COVID-19 in various countries during the initial six months of the pandemic, from January 2020 to July 2020. Using a dataset sourced from Kaggle, the study explores the spread and impact of the virus across different nations. The key objectives include identifying the most affected countries, examining testing efforts, and evaluating COVID-19's impact relative to population size.

2. Project Questions

The analysis aims to answer the following questions:

  1. Which countries had the highest number of deaths due to COVID-19?
  2. Which countries recorded the highest number of positive cases relative to tests conducted?
  3. Which countries demonstrated the best testing efforts relative to their population?
  4. Which countries were most and least affected relative to their population?

3. Understanding Data

3.1 Dataset Overview

The dataset consists of 10,903 rows and 14 columns, including variables such as positive cases, deaths, recoveries, total tests, and active cases. Importantly, the dataset contains no missing values or duplicates.

3.2 Key Variables

  • positive: Cumulative number of positive cases reported.
  • death: Cumulative number of deaths reported.
  • total_tested: Total number of COVID-19 tests conducted.
  • daily_positive: Number of positive cases reported daily.
  • daily_tetsted: Number of tests conducted daily.
  • active: Number of active cases on a given day.

4. Data Cleaning

To prepare the dataset for analysis, the following steps were performed:

  • Standardized column names.
  • Converted the date column to datetime format.
  • Filtered data to retain only country-level records (excluding state/provincial data).
  • Created two subsets:
    1. Cumulative data (Total cases, deaths, and recoveries per country).
    2. Daily data (New cases, new tests, and active cases per day).
