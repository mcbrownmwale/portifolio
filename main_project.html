Predicting Insurance Costs

Tools: Python [ Pandas, Numpy, Seaborn, Scikit Learn]

Introduction

For this project we looked at the Medical Cost Dataset from kaggle. This dataset contains information on individual medical insurance bills. Each bill is associated with some demographic and personal characteristics of the person who received it.

Problem Statement

For our special problem, we were interested in how these different characteristics relate to the total medical cost. We constructed the best possible predictive model for the cost given some information about the patient.

Summary of Results

  1. A one year increase in a person's age would result into 5% increase in insurance charges holding other features constant.
  2. If a person is a smoker, his or her insurance charges would increase by 822% holding other features constant.
