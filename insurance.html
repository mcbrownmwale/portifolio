Predicting Insurance Costs

Tools: Python [ Pandas, Numpy, Seaborn, Scikit Learn]

1. Introduction

This project aims to predict individual medical insurance costs using the Medical Costs dataset from Kaggle. The dataset contains demographic and personal characteristics associated with insurance charges. The primary objective is to build a predictive model that accurately estimates these costs based on available patient data.

1.1 Problem Statement

For our special problem, we were interested in how these different characteristics relate to the total medical cost. We constructed the best possible predictive model for the cost given some information about the patient.

2. Methodology

2.1 Data Exploration and Preparation

Checking for Missing Data Points

A review of the dataset confirmed that there were no missing values, ensuring data completeness

Identifying and Removing Duplicates

One duplicate record was found and removed to maintain data integrity.

Correlation Analysis

A correlation Analysis revealed that:

  • Age and insurance charges had a moderate positive correlation (~30%).
  • BMI and number of children had weak correlations with insurance charges.
  • The distribution of charges was right-skewed, prompting a log transformation to normalize the data.

Feature Engineering

  • A new column, log_charges, was created by applying a logarithmic transformation to the charges column.
  • The categorical smoker variable was converted into a binary format (1 for smokers, 0 for non-smokers).

2.2 Model Building

Data Splitting

  • The dataset was split into training (80%) and testing (20%) sets.
  • Features selected for the model: age and smoker_yes.

Selection and Training

A Linear Regression model was trained using the selected features.

Model Evaluation

  • R-Squared value (Goodness of fit)
    • Training data: 72.19%
    • Testing data: 79.97%
  • The model explains a significant portion of variance in insurance costs, indicating a strong predictive capability.

2.3 Interpretation of Coefficients

  • Age: A one-year increase in age results in a 5% increase in insurance charges.
  • Smoking: Being a smoker increases insurance charges by 822%, highlighting the impact of smoking on insurance costs.

3. Conclusion

This project successfully developed a regression model that predicts insurance costs based on age and smoking status. While the model performs well, further improvements can be achieved by incorporating additional features such as BMI and region. Additionally, exploring alternative machine learning models could enhance predictive performance.
